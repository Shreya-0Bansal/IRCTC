IRCTC C++ Project - Comprehensive Train Reservation System
Welcome to the IRCTC C++ Project, a comprehensive train reservation system designed to provide all essential services, including admin authentication, train booking, updating bookings, and ticket cancellation.

Features
Admin Authentication: Secure password authentication for admin access.
Train Booking: Allows users to book train tickets seamlessly.
Update Bookings: Enables users to modify existing bookings.
Cancel Tickets: Facilitates the cancellation of booked tickets.
